-------------------------------------------------------------------------------------------------------------------
Файлы с тестовыми данными:
        test_get_task.csv - тестирование команд get
 test_just_additional.csv - тестирование команд add
     test_just_remove.csv - тестирование команд remove
     test_just_update.csv - тестирование команд update
  test_mix_operations.csv - смешенное тестирование( содержит разные типы команд в разной последновательности)
-------------------------------------------------------------------------------------------------------------------
все команды разделены ','. Регистр не важен.

Последовательность первых 2х столбцов ВАЖНА!, остальные могут быть в произвольном порядке(актуально для команд add и update)
1й столбец - название тестируемых команд(add, UPDATE, REMOVE, get, clone)
2й столбец:
    + для команд add, clone и UPDATE - тип задачи(TASK, EPIC, SUB_TASK)
    + для команд REMOVE и get  - типы задачи, и дополнительные команды

для команды REMOVE:
           all - очистить таск-менеджер
       allepic - удалить все задачи
       alltask - удалить все эпики
    allsubtask - удалить все подзадачи

для команды get:
           all - получить все задачи, эпики и подзадачи
       allepic - получить все задачи
       alltask - получить все эпики
    allsubtask - получить все подзадачи
   epicSubtask - получить подзадачи эпика (здесь фантазия спасовала - дурацкая команда вышка)
